# Sensor-ball
Проект датчика качества воздуха с дисплеем.

Устройство построен на базе модуле ESP32-s3.

Состав устройства:
1. модуль ESP32-s3 supermini
2. Круглый дисплей 1,5" ST77916 разрешение 360*360
3. Датчик углекисорго газа scd40 на шине i2c
4. Датчик освещения tsl2561 на шине i2c
5. i2s микрофон INMP441
6. Корпус выполненный методом 3D-печати

Функции:
1. отображение содержания частиц CO2 в воздухе
2. отображение текущей температуры и влажности
3. Определение уровня шума в помещении+графический эквалайзер(микрофон в конфигурации Esphome откалиброван по прибору UNI-T UT353. Спасибо 
4. Автоматическая регулировка яркости дисплея в зависимости от уровня освещенности. Измерение уровня освещенности на поверхности (Сенсор в конфигурации Esphome откалиброван по прибору MOBIPLUS WT81B).
   Графическая часть выполнена на LVGL.
